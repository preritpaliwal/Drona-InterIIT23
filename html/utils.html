utils module¶

utils.getBytes(value)¶

Function to return 8-bit LSB and MSB as a list of the given value

utils.getDec(lsb, msb, base=256)¶

Function to return back the unsigned number given LSB and MSB. The base is default at 256 but can be passed as per requirement

utils.getSignedDec(lsb, msb)¶

Function to return signed number given the LSB and MSB
